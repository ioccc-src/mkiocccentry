# pr

libpr - stdio helper library


# To install

```sh
make clobber all
sudo make install clobber
```


# To use


## In your C code:

```c
#include <pr.h>
```


## To compile and link:

```
cc -I/usr/local/include prog.c -L/usr/local/lib -lpr -o prog
```


# See also:

```
man 3 pr
```

## To see prior to installation:

```
man man/man3/pr.3
```
